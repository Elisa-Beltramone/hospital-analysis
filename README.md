# Hospital Control Room

Dashboard de analítica hospitalaria desarrollado con Streamlit.

## Funcionalidades

- KPIs operativos
- Predicción de demanda 24h
- Alertas de saturación
- Análisis de diagnósticos
- Análisis por médico
- Heatmaps temporales

## Ejecutar

```bash
pip install -r requirements.txt
streamlit run app.py
